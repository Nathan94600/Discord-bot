# To start the bot :

- Download the TypeScript module :

```bash
npm i -g typescript
```

- Install the necessary modules for the bot

```bash
npm i
```

- Create a file named `config.json` in the `src` directory and add your token inside, like this :

```json
{ "token": "YOUR_TOKEN" }
```

- Build the bot :

```bash
npm run build
```

- Start the bot :

```bash
npm run start
```
